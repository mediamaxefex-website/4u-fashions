# Green Zone Pool Resort HTML Site

This is the static HTML version of the Green Zone Pool Resort website.

## Files

- `index.html`
- `about.html`
- `stay.html`
- `experiences.html`
- `events.html`
- `gallery.html`
- `contact.html`

## Open the website

Open `index.html` in your browser.

## Main edit points

### Contact details and social links

Edit:

- `contact.html`
- Footer areas in each page if you want the same real details everywhere

### Google Map embed

In `contact.html`, find this line:

`<iframe class="map-frame" src="" ...></iframe>`

Paste your Google Maps embed link inside `src=""`.

### Reviews

Edit the review cards in:

- `index.html`
- `contact.html`

### Images

All photos are in:

- `assets/images`

## Quick edit suggestion

Use VS Code. Right-click `index.html` and open with browser, or use a live server extension if you want easier preview while editing.
